# 목적

# 지원 브라우저, 의존성
* Browser : Chrome, Safari, Firefox, IE10+

# 준비
## OSX
1. brew install npm
1. cd $PROJECT_HOME
1. npm install
1. npm intstall -g nodemon 


# 실행방법
```
nodemon app.js
```
